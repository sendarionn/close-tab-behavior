# close-tab-behavior

アクティブなタブを閉じた後の移動先を変更する Chrome 拡張機能

コードは Codex を使用して作成

## 機能

- 直近でアクティブだったタブに移動
- 最も右のタブに移動
- 最も左のタブに移動
- 固定タブを移動先から除外
- ウィンドウごとにタブの履歴を管理
- 変更した設定を Chrome Sync に自動保存

## インストール

1. このリポジトリをダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパー モード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」を選択
5. ダウンロードしたフォルダを指定

## 設定

1. `chrome://extensions/` を開く
2. `close-tab-behavior` の「詳細」を選択
3. 「拡張機能のオプション」を選択
4. 移動先または固定タブの設定を変更

変更した設定は自動で保存される

移動先が変わるのはアクティブなタブを閉じたときだけ

バックグラウンドのタブを閉じても表示中のタブは変わらない
